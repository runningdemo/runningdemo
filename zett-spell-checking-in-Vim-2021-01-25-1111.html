Spell checking in Vim

Keywords: vim,spell check,
Date: 2021-01-25

Turn on spell check

set spell
set spelllang=en_us

Jump between the spell errors

[s: next spell error

]s: previous spell error

Specify the spell file

set spellfile=$HOME/.vim-spell-en.utf-8.add

Make the spell error pop (if you want to)

hi clear SpellBad
hi SpellBad ctermbg=red ctermfg=white

Opt-in project specific spell checking and word completion

In you own .vimrc, turn on the exrc option, and put a .exrc file under the project folder.

" .exrc

set spelllang=en_us
set spellfile=en.utf-8.add

" Disable unsafe commands.
" http://andrew.stwrt.ca/posts/project-specific-vimrc/
set secure
