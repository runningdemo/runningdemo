Convert my notes from markdown to html

Keywords: vimwiki,site-generator,notes
Date: 2021-01-09

I write my note in vim using vimwiki. And generating html files from the note markdown files and deploy to github pages to make it accessbbile from the web.

Prerequisites

Install some tools: pandoc, ripgrep

Setup

Create a website folder to put all the generated html files

Read all the note files in vimwiki folder

Convert them to html file with pandoc

Move them to the site director

Build the index.html

Generate a entry point to list all the notes.

Automatically run the build script when files update

Run this command in the build script directory.

while [[ 1 ]]; do inotifywait -e modify ./src/* ~/vimwiki/*.md; ./src/b.sh; done;

Setup a local server

python3 -m http.server 4000

Include live.js for automatically refresh webpage when content changed

Put this inside the html template file:

<script src="https://livejs.com/live.js"></script>
