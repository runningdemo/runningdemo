Highlight cursor in arch linux

Keywords: remote,meeting
Date: 2021-01-28

Make the cursor larger

Change the cursor size in .Xresources file and reload the settings with command xrdb ~/.Xresources

Xcursor.size: 64

After reload the configuration, the cursor size won’t update in the existing application windows e.g. Firefox, in this case you need to quit the application first then launch it again, you can see the cursor size updated.
